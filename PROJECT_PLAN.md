# HIFU Dash Project Plan

HIFU Dash 可以先從一個 2 到 4 週內能上架測試的 MVP 開始，不必一開始就做完整 Geometry Dash。第一版目標是讓使用者在最短時間內理解核心玩法，也讓醫學會、衛教、投資人簡報與展場互動都有可玩的版本。

## 第一階段：MVP（2 到 4 週）

核心玩法只做四件事：

- 自動往右跑
- 點擊跳躍
- 碰到障礙死亡
- 通關終點

MVP 功能範圍：

| 功能 | 是否納入 |
| --- | --- |
| 自動跑 | Yes |
| 跳躍 | Yes |
| 障礙物 | Yes |
| 死亡重來 | Yes |
| 音樂同步 | No |
| 飛船模式 | No |
| 重力翻轉 | No |
| 編輯器 | No |

這樣的範圍 1 位 Unity 工程師即可完成第一版。

## Unity 技術架構

推薦：

- Engine: Unity 6
- Language: C#

建議架構：

```text
GameManager
|
+-- PlayerController
+-- ObstacleManager
+-- LevelManager
+-- UIManager
+-- AudioManager
```

### PlayerController

負責：

- 跳躍
- 碰撞
- 死亡
- 重生

主要方法：

```csharp
Jump()
Die()
Respawn()
```

### ObstacleManager

負責：

- 生成尖刺
- 生成血管、神經、器官等障礙
- 管理移動障礙
- 清除離開畫面的物件

主要方法：

```csharp
SpawnObstacle()
DestroyObstacle()
```

### LevelManager

負責：

- 第 1 關
- 第 2 關
- 第 3 關
- 終點判定
- 關卡進度

## 美術需求

MVP 階段不需要大量美術。

最小可行美術：

- 玩家：藍色超音波焦點，或先用簡單發光圓形
- 障礙：三角形尖刺、血管線條、器官色塊
- 背景：藍色、紫色、黑色漸層
- UI：PLAY、YOU DIED、LEVEL COMPLETE、LEVEL SELECT

正式 HIFU 版可以再把障礙換成：

- 血管
- 神經
- 器官
- 腫瘤目標

## 第二階段：關卡編輯器

Geometry Dash 成功的關鍵不是只有遊戲本身，而是玩家可以自己做關卡。

第二階段可以加入簡易編輯器：

```text
----------------
尖刺
平台
傳送門
血管
神經
器官
腫瘤
----------------
拖曳放置
```

操作體驗可以像 PowerPoint：

- 點選物件
- 拖曳放置
- 調整位置
- 儲存關卡
- 測試遊玩

資料儲存格式範例：

```json
{
  "objects": [
    {
      "type": "spike",
      "x": 100
    },
    {
      "type": "platform",
      "x": 250
    },
    {
      "type": "tumor",
      "x": 480
    }
  ]
}
```

## 第三階段：AI 自動生成關卡

第三階段最有差異化的是 AI 自動生成關卡。

使用者上傳音樂後，系統可以：

1. 分析 BPM
2. 找節拍
3. 自動放障礙
4. 生成可玩的關卡

流程：

```text
MP3
 |
Beat Detection
 |
產生節奏點
 |
生成障礙
 |
生成關卡
```

節奏對應玩法範例：

| 音樂事件 | 關卡事件 |
| --- | --- |
| 鼓點 | 單一尖刺 |
| 連續鼓點 | 連續障礙 |
| 副歌 | 高密度障礙 |
| Drop | 腫瘤目標或特殊效果 |

Unity 內可串接：

- OpenAI
- Claude
- Gemini

用途是協助生成關卡參數，而不是即時控制遊戲。

## HIFU Dash 定位

HIFU Dash 的玩法跟 Geometry Dash 一樣直覺，但主題改成醫療展示：

- 主角：超音波焦點
- 障礙：血管、神經、器官
- 目標：打到腫瘤
- 失敗條件：傷到正常組織

適合用途：

- 醫學會展示
- 醫院衛教
- 投資人簡報
- 展場互動

這會比單純 PowerPoint 更容易讓觀眾理解 HIFU 的核心價值：精準命中目標，同時保護正常組織。

## 預估成本與時程

以 1 位 Unity 工程師加 1 位美術估算：

| 範圍 | 預估費用（台幣） | 預估時間 |
| --- | ---: | --- |
| MVP | 10 到 20 萬 | 2 到 4 週 |
| 關卡編輯器 | 30 到 50 萬 | 1 到 2 個月 |
| AI 自動生成關卡 | 50 到 100 萬 | 2 到 3 個月 |

2 到 3 個月內可以做出一個能讓投資人、醫師或客戶實際遊玩的版本。
